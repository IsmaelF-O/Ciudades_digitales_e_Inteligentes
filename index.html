📊 Acerca del Proyecto

Este proyecto desarrolla un sistema integral de limpieza, procesamiento y análisis bibliométrico de información científica obtenida desde la base de datos Scopus. Su propósito es transformar grandes volúmenes de datos sin procesar en información estructurada y análisis significativos.

La investigación se centra en el análisis de publicaciones relacionadas con ciudades inteligentes, transformación digital urbana y geopolítica del conocimiento en Iberoamérica durante los últimos años, permitiendo identificar tendencias emergentes y actores clave en el desarrollo de estas temáticas.

El sistema implementado elimina registros duplicados, corrige valores faltantes, normaliza palabras clave, transforma variables numéricas y organiza la información en estructuras de datos eficientes para facilitar análisis profundos y visualizaciones significativas.

🎯 Objetivo Principal

🧹 Limpieza de Datos

Preparación rigurosa de datos bibliométricos eliminando inconsistencias, duplicados y valores faltantes para garantizar calidad analítica.

🔄 Transformación

Estructuración y reorganización de información en formatos útiles y accesibles mediante procesamiento avanzado de datos.

📈 Análisis

Identificación de patrones, tendencias, correlaciones y relaciones significativas en la investigación científica.

📊 Visualización

Presentación clara, intuitiva e interactiva de hallazgos para facilitar la comprensión de datos complejos.

💻 Tecnologías y Herramientas

🐍 Python

Lenguaje principal del proyecto por su flexibilidad, facilidad de aprendizaje y amplio ecosistema de herramientas para ciencia de datos, análisis estadístico y minería de información.

📦 Pandas

Biblioteca principal para manipulación y análisis de datos. Permite cargar archivos CSV desde Scopus y organizar información en DataFrames para procesamiento eficiente.

🔢 NumPy

Herramientas para procesamiento numérico de alto rendimiento. Realiza cálculos estadísticos, operaciones matemáticas y transformaciones de datos necesarias para análisis rigurosos.

📉 Matplotlib

Generación de visualizaciones científicas que representan gráficamente la evolución temporal, tendencias de investigación e indicadores bibliométricos clave.

🔍 Collections

Módulo mediante la clase Counter para realizar conteos automáticos de frecuencia. Fundamental para identificar palabras clave y temas más utilizados en la investigación.

📝 Regular Expressions

Biblioteca Re para limpieza y normalización de texto. Elimina caracteres especiales, corrige inconsistencias en palabras clave y estandariza formatos de información.

✨ Funcionalidades Implementadas

  • Limpieza y normalización de datos bibliográficos
  • Eliminación de registros duplicados
  • Tratamiento de valores faltantes
  • Procesamiento y limpieza de palabras clave
  • Identificación de temas de investigación predominantes
  • Análisis de autores más productivos
  • Identificación de autores más citados
  • Análisis de revistas con mayor producción científica
  • Estudio de tendencias temporales de publicación
  • Detección preliminar de temas emergentes
  • Exportación automática de resultados en formato CSV
  • Generación de visualizaciones bibliométricas

🚀 Desarrollo Futuro

Las siguientes etapas del proyecto contemplan la incorporación de técnicas avanzadas de análisis bibliométrico e inteligencia artificial:

🧠 LDA - Modelado de Temas

Latent Dirichlet Allocation para descubrimiento automático de temas latentes en textos científicos.

🤖 BERTopic

Técnicas avanzadas de procesamiento de lenguaje natural para descubrimiento dinámico de temas.

📊 Algoritmo de Kleinberg

Detección de explosiones temáticas e identificación de períodos de actividad investigativa intensa.

📈 Dashboards Interactivos

Visualizaciones dinámicas e interactivas utilizando Plotly y JavaScript para exploración de datos.

🔮 Análisis Predictivo

Modelos de machine learning para predicción de tendencias científicas futuras.

🗣️ Procesamiento NLP

Integración completa de técnicas avanzadas de procesamiento de lenguaje natural.
